Overview

七牛 CDN 日志文件集合

该类存储一个 Hash，Key 为日志文件所在域名，而 Value 则为一个数组， 每个数组都是一个 QiniuNg::CDN::LogFile 对象，用于保存一个日志文件。

对本类可以直接调用 Enumerable 内的方法， 用于对该 Hash 的数据进行遍历
